# PlapyPlane
 
Hyper Casual Game ( Hiperbasit oyun)
Oynanış için aşağıdaki bağlantıdan oynayabilirsiniz.
if you are playing -> https://play.unity.com/en/games/6415c785-220d-4355-9a1b-0dc0d0a97180/webgl-builds

Oyun içi tuşlar
"Space" zımplama
"R" yeniden başlat
"M" menüye dön
"P" oyunu durdur

key config
Space is jump
R is restart
M is backt o menu
P is pause
